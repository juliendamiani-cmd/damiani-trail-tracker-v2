DAMIANI TRAIL TRACKER V35 — SUIVI ROBUSTE ET HORS CONNEXION

La base Firebase du projet est préconfigurée.
Code de suivi par défaut : JULIEN-UT4M-50B-2026

UTILISATION COUREUR
1. Ouvrir l’application et choisir le mode coureur.
2. Dans « Synchronisation avec mes proches », vérifier le code de suivi puis activer la synchronisation.
3. Appuyer sur « Inviter mes proches ».
4. Partager le lien proposé. Il prend la forme : ?suivi=CODE

UTILISATION PROCHES
1. Ouvrir le lien reçu.
2. L’application s’ouvre automatiquement en mode Proches, sans écran de choix ni configuration technique.
3. Le code de suivi est mémorisé sur l’appareil.
4. Les données sont actualisées automatiquement environ toutes les 5 secondes.

ACQUIS CONSERVÉS DE LA V34
- Lien simplifié au format ?suivi=CODE.
- Ouverture automatique en mode Proches.
- Mémorisation locale du code de suivi.
- Bouton principal « Inviter mes proches » côté coureur.
- Interface Proches allégée : les réglages techniques et le bouton de sortie sont masqués lors d’une ouverture par lien.
- Compatibilité maintenue avec les anciens liens contenant mode=family, firebase=... et sync=....
- Le partage depuis le mode Proches conserve le lien de suivi simplifié.

MESSAGERIE
- Les proches peuvent saisir leur prénom et envoyer un message de 300 caractères maximum.
- Julien reçoit les messages automatiquement lorsqu’il a accès à Internet.
- La réinitialisation complète efface aussi les messages de la course.

SÉCURITÉ
Les règles Firebase publiques permettent la lecture et l’écriture à toute personne possédant le lien. Ne partager le lien qu’avec des personnes de confiance. Après la course, désactiver les règles publiques ou supprimer les données.
